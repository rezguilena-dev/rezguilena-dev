##  Projets réalisés :
###  [Network Intrusion Detection System (NIDS)](https://github.com/rezguilena-dev/Network-Intrusion-Detection-System)
Développement d'un moteur de détection d'intrusions réseau en temps réel couplé à un tableau de bord analytique.
* **Contexte** : Projet personnel 
* **Défis système & réseau** : Architecture d'un laboratoire virtualisé (gestion du pont réseau hôte/VM) et contournement des contraintes OS lors des tests(empêchant l'envoi des trames ARP falsifiées par le kernel Windows).
* **Défis de détection:** : gestion de l'alerte fatigue, calibrage du seuil d'entropie de Shannon face au bruit légitime (ex: pings Windows à haute entropie) pour minimiser les faux positifs sans compromettre la détection des véritables attaques. 
* **Outils & Stack** : Python ,Scapy,Wireshark, Streamlit
